# Sistema CEPA · v2

Rediseño del Sistema CEPA (Centro de Psicología Aplicada, Universidad de Talca) construido con **Vite + React + TypeScript + Tailwind v4 + shadcn/ui**.

## Vista en vivo

https://dario-kreante.github.io/cepa-preview-v2/

## Stack

- React 19 + TypeScript
- Vite + Tailwind v4
- shadcn/ui (Radix primitives + CVA)
- lucide-react para iconos
- recharts para gráficos

## Desarrollo

```bash
npm install
npm run dev       # servidor local
npm run build     # build a docs/ (usado por GitHub Pages)
```

## Módulos

Dashboard · Ingresos y pacientes · Licencias médicas · Gestión de fármacos · Controles médicos · Seguimiento EPT · Reintegro laboral · Auditoría · Agendamiento · Reportería.
